REST API Endpoints

Chat & Sessions
  • POST /api/chat - Stream responses with model selection
  • GET /api/sessions - Session history with filtering
  • POST /api/sessions - Create new chat session
  • PUT /api/sessions/[id] - Update session details
Message Management
  • POST /api/stars - Star/unstar messages
  • PUT /api/sessions/messages/[messageId]/tags - Update tags
  • GET /api/tags - Fetch tags with statistics
